# Building 
* Required Dependencies 
	* [MFEM](https://github.com/mfem/mfem) built with MPI, [Hypre](https://github.com/hypre-space/hypre), [Metis](https://github.com/mfem/tpls/blob/gh-pages/metis-4.0.3.tar.gz) and optionally Suitesparse and Libunwind are useful 
	* [igraph](https://github.com/igraph/igraph.git) 
	* [Kokkos mdspan](https://github.com/kokkos/mdspan)
	* [yaml-cpp](https://github.com/jbeder/yaml-cpp.git)
	* [sol2](https://github.com/ThePhD/sol2.git) and some version of Lua5.x 
* Optional Dependencies 
	* [GoogleTest](https://github.com/google/googletest.git)
